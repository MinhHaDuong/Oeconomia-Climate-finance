# RDJ4HSS Author Guidelines — Key specs for data paper submission

Source: https://platform.openjournals.nl/RDJHSS/about/submissions
(downloaded 2026-03-26 from libraryFiles/downloadPublic/211)

## Word limits
- Data paper: **2,500 words maximum** (no exclusions stated)
- Abstract: **200 words maximum**

## Submission format
- Text as a **Word file** (.docx), using default Word styles
- Figures/tables as **separate files**, not embedded in the paper
- File names include type and number: e.g., Figure1.png, Table1.png
- Link (DOI) to archived dataset in a trusted repository

## Structure (may deviate slightly)
- Title, author name(s), affiliation(s)
- Abstract
- Keywords (2-8, **lowercase**)
- Related dataset(s) with DOI
- Introduction
- Problem (optional, can merge with Introduction)
- Methods
- Data (general characteristics, technical details, documentation)
- Concluding remarks
- Acknowledgements (**separate file**)
- References

## Formatting
- Title: 16pt bold, followed by author name, affiliation, email
- Body: 11pt roman font
- Pages numbered consecutively
- British or American English, consistent throughout
- Footnotes (not endnotes), sparingly
- Remove internal cross-references and reference manager links

## References
- **APA 7th edition**
- DOIs formatted as URLs, no trailing period
- Every text citation in reference list and vice versa

## Illustrations
- JPG, TIF, or SVG (high resolution, 1000x1000px min, 300 dpi preferred)
- PNG for text-heavy images (tables, charts)
- No GIF, BMP, or embedded Word/PDF images
- Sequential numbering: Figure 1, Table 1, Figure 2, Table 2, etc.
- Each with a short caption

## Review
- Paper reviewed by peer reviewers AND data specialists
- Data reviewed separately against **FAIR principles**
- Data must pass quality criteria or paper cannot be published
